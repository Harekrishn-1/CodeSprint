# 🚀 CodeSprint 

CodeSprint is a full-stack coding platform built using the MERN stack, inspired by LeetCode. It allows users to practice coding problems, track progress, and improve problem-solving skills.

---

## 🌟 Features

* 👨‍💻 Solve coding problems (DSA focused)
* 🧠 Multiple difficulty levels (Easy, Medium, Hard)
* 📝 Code editor for writing and testing solutions
* 📊 Track your progress and performance
* 🔐 User authentication (Login/Signup)
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Tailwind CSS / CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Harekrishn-1/CodeSprint.git
cd CodeSprint
```

### 2️⃣ Install dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3️⃣ Run the project

```bash
# Run backend
cd backend
npm start

# Run frontend
cd frontend
npm start
```
